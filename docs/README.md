- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software required

### **Installation**

1. **Clone the repository**

   ```bash
   git clone https://github.com/nzaoo/nzao_card.git
   ```

2. **Navigate to project directory**

   ```bash
   cd nzao_card
   ```

3. **Open in browser**

   ```bash

   # On macOS
   open src/index.html
   ```
